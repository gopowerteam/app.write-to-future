## 简介

本项目使用 Vue + Vuetify + Nuxt + Nestjs + Typescript 构建

## 安装

```bash
$ npm install
```

## 启动

```bash
# development
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
# $ npm run test:e2e

# test coverage
# $ npm run test:cov
```

## 端口

```
localhost:3000          web页面端口
localhost:3000/api/**   服务端api接口
localhost:3000/doc      服务端文档接口
localhost:3000/graphql  graphql测试接口
```
